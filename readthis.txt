- QuizForge

---------------------------------------------
4) LOGIC REQUIREMENTS
---------------------------------------------
- Uploading file updates profile count immediately.
- Deleted files reduce count immediately.
- Bookmarks persist and display later.
- Sessions and activity history update based on quiz/game actions.
- Quiz setting toggle affects flow behavior.
- Timer setting 0 => no timer.
- AI model selection affects generation service routing.

---------------------------------------------
5) AI INTEGRATION REQUIREMENTS
---------------------------------------------
Gemini integration should provide:
- quiz generation
- reviewer notes
- flashcards
- AI chat

Service contract suggestion:
- generateQuizFromText(String content, QuizConfig config)
- generateReviewerNotes(String content)
- generateFlashcards(String content)
- chatWithContext(String content, String userPrompt)

Implement:
- loading, success, retry, error states
- asynchronous requests
- educational tone and concise explanation

Optional model strategy:
- selectedModel == 'Grok' => use xAI endpoint if key available
- selectedModel == 'Google Gemini' => use Gemini endpoint
- fallback to user-selected available provider
